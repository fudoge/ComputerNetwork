
# Client
## Client : Terminal
- 메시지를 입력받아 전달(계속해서)

## Client: Display
- 유닉스 클라이언트(Clinet: Terminal)로부터 비동기적으로 입력받은 데이터를 서버에전송

# Server
## main(대기실)
- 서버가 각 클라이언트들에게 메뉴 던져줌
1. 서버 목록(현재 서버 현황)
2. 서버 접속 (2 <방번호>) -> 그 방으로 접속시켜줌
3. 클라이언트 연결 종료
## 각 채팅방 스레드
- 클라이언트로부터 메시지가 왔는지 감지
- 채팅방 내에 있는 다른 사용자들에게 전달
- "quit" 입력받을 시, 메인으로 보냄
